# Tris

Tris is a simple tic-tac-toe game made in C (c23).

## Installation

Download the 'tris' file on Linux system or the 'tris.exe' file on Windows.

Execute from terminal or double click to start.

## Purpose

Tris is a project aimed at testing and mastering a number of basic instructions in C, implementing the code DRY and compact while also ensuring readability.

Tris is only 210 lines long and includes/ensures:
* Structures
* Functions
* Cycles
* Arrays & pointers
* Control flow
* Handling errors
* Handling user input
* Buffer management
* ANSI codes

## Features

In Tris you can:
* play against the CPU;
* play against a friend;
* insert your(s) name(s);
* play multiple times without restarting.
